About Fake News Detection System

The Fake News Detection System is a web-based application designed to identify whether a given news article is genuine or misleading. With the rapid spread of information through digital platforms, verifying the authenticity of news has become a critical challenge. This project aims to address that challenge using Machine Learning and Artificial Intelligence techniques.

Project Objective

The primary objective of this project is to help users analyse news content and determine its credibility. The system provides an easy-to-use interface where users can paste a news article and receive a classification result along with a confidence score.

Technology Used

  • Python & Flask for backend development
  • Machine Learning models: Logistic Regression, Random Forest, Gradient Boosting and Decision Tree
  • TF-IDF Vectorization for text feature extraction
  • Google Gemini AI for additional verification and explanation
  • HTML & CSS for user interface design

How the System Works

When a user submits a news article, the text is first cleaned and processed. The processed text is then analysed by multiple machine learning models trained on labelled news datasets. Each model provides its own prediction.

In addition to machine learning models, the system uses Google Gemini AI to further verify the news content. The final decision is made using an ensemble voting mechanism, and a confidence score is calculated based on the combined results.

Key Features

  • Multiple machine learning models for improved accuracy
  • AI-based explanation for better transparency
  • User-friendly and clean interface
  • Reliable and stable performance without external dependencies

Conclusion

This project demonstrates how machine learning and artificial intelligence can be effectively combined to tackle real-world problems such as fake news detection. The system is intended for educational and academic purposes and highlights the practical application of data science concepts.
